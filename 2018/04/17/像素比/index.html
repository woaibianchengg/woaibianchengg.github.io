像素比

发表于 2018-04-17 |

如果想要在移动设备上显示 1 px 带背景颜色的边框的话，可以通过以下几个方面去实现。

1：通过transform来变形

.item {
position: relative;
}
.item:after {
content: ‘’;
display: block;
position: absolute;
width: 100%;
left: 0;
bottom: 0;
height: 1px;
background-color: #c8c7cc;
-webkit-transform: scaleY(0.5);
transform: scaleY(0.5);
//房租鼠标点击
pointer-events: none;
}

2:通过stylus预编译器对设备像素比进行检测

首先，新建一个 mixin.styl

border-1px($color)
position: relative
&:after
display: block
position: absolute
left: 0
bottom: 0
width: 100%
border-top:1px solid $color
content: ‘ ‘
//房租鼠标点击
pointer-events: none;

然后再建立一个base.styl,来根据设备dpr进行缩放

@media (-webkit-min-device-pixel-ratio: 1.5),(min-device-pixel-ratio: 1.5)
.border-1px
&::after
-webkit-transform: scaleY(0.7)
transform: scaleY(0.7)

@media (-webkit-min-device-pixel-ratio: 2),(min-device-pixel-ratio: 2)
.border-1px
&::after
-webkit-transform: scaleY(0.5)
transform: scaleY(0.5)

为了方便引用，新建一个index.styl文件统一引入所有styl文件

@import ‘./mixin’
@import ‘./base’

然后以vue项目为例，在main.js中引入

import ‘./common/stylus/index.styl’

随后要在样式的元素中添加 border-1px 类名和在其样式上添加 border-1px(clor颜色值);就可以实现边框在移动端上以1px显示了
