vue-transition

发表于 2018-04-18

Vue 给我们提供了一个关于动画过渡的封装组件，根据官网的例子来看，可以这样去理解

v-enter: 定义进入过渡的开始状态。在元素被插入时生效，在下一个帧移除。
v-enter-active: 定义进入过渡的结束状态。在元素被插入时生效，在 transition/animation 完成之后移除。
v-leave: 定义离开过渡的开始状态。在离开过渡被触发时生效，在下一个帧移除。
v-leave-active: 定义离开过渡的结束状态。在离开过渡被触发时生效，在 transition/animation 完成之后移除。

像素比

发表于 2018-04-17

如果想要在移动设备上显示 1 px 带背景颜色的边框的话，可以通过以下几个方面去实现。

1：通过transform来变形

.item {
position: relative;
}
.item:after {
content: ‘’;
display: block;
position: absolute;
width: 100%;
left: 0;
bottom: 0;
height: 1px;
background-color: #c8c7cc;
-webkit-transform: scaleY(0.5);
transform: scaleY(0.5);
//房租鼠标点击
pointer-events: none;
}

2:通过stylus预编译器对设备像素比进行检测

首先，新建一个 mixin.styl

border-1px($color)
position: relative
&:after
display: block
position: absolute
left: 0
bottom: 0
width: 100%
border-top:1px solid $color
content: ‘ ‘
//房租鼠标点击
pointer-events: none;

然后再建立一个base.styl,来根据设备dpr进行缩放

@media (-webkit-min-device-pixel-ratio: 1.5),(min-device-pixel-ratio: 1.5)
.border-1px
&::after
-webkit-transform: scaleY(0.7)
transform: scaleY(0.7)

@media (-webkit-min-device-pixel-ratio: 2),(min-device-pixel-ratio: 2)
.border-1px
&::after
-webkit-transform: scaleY(0.5)
transform: scaleY(0.5)

为了方便引用，新建一个index.styl文件统一引入所有styl文件

@import ‘./mixin’
@import ‘./base’

然后以vue项目为例，在main.js中引入

import ‘./common/stylus/index.styl’

随后要在样式的元素中添加 border-1px 类名和在其样式上添加 border-1px(clor颜色值);就可以实现边框在移动端上以1px显示了

发表于 2018-04-16

在最新版本的vue-cli 没有’dev-server.js’以及’dev-client.js’两个文件之后，如果想对饿了吗数据进行获取，应该这样去操作！
在新版本的build文件夹内的webpack.dev.conf.js文件夹内，进行相关的数据获取。

//首先
const express = require(‘express’)
const app = express()
var appData = require(‘../data.json’)
var seller = appData.seller
var goods = appData.goods
var ratings = appData.ratings

//添加路由
var apiRoutes = express.Router()
app.use(‘/api’, apiRoutes)

然后找到devServer
在对象里添加
before(app) {
app.get(‘/api/seller’, (req, res) => {
//你的Json内容
res.json({
errno: 0,
data: seller
})
}),
}
