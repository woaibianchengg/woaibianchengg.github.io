canvas

发表于 2018-05-26

drawImage虽然名字看起来像是在画布Canvas上绘制图像，但是它还可以绘制视屏等。
drawImage有三种不同的使用方式：
　　
drawImage(img,x,y);
drawImage(img,x,y,width,height);
drawImage(img,sx,sy,swidth,sheight,x,y,width,height);
　　
drawImage(img,x,y)：img指定图像，x和y指定图像在画布Canvas上的坐标。注意是相对于画布而言的，不是屏幕。
　　
drawImage(img,x,y,width,height)：width和height是指定图像的大小，通过指定该值可以将图像进行缩放和放大。
　　
drawImage(img,sx,sy,swidth,sheight,x,y,width,height)：使用该方法可以在图像上进行裁剪，并将裁剪的区域显示在指定的位置。其中sx和sy指定裁剪区域的起始坐标，该坐标是相对于图片的，不是画布。swidth和sheight是裁剪区域的大小。

vue-transition

发表于 2018-04-18

Vue 给我们提供了一个关于动画过渡的封装组件，根据官网的例子来看，可以这样去理解

v-enter: 定义进入过渡的开始状态。在元素被插入时生效，在下一个帧移除。
v-enter-active: 定义进入过渡的结束状态。在元素被插入时生效，在 transition/animation 完成之后移除。
v-leave: 定义离开过渡的开始状态。在离开过渡被触发时生效，在下一个帧移除。
v-leave-active: 定义离开过渡的结束状态。在离开过渡被触发时生效，在 transition/animation 完成之后移除。

像素比

发表于 2018-04-17

如果想要在移动设备上显示 1 px 带背景颜色的边框的话，可以通过以下几个方面去实现。

1：通过transform来变形

.item {
position: relative;
}
.item:after {
content: ‘’;
display: block;
position: absolute;
width: 100%;
left: 0;
bottom: 0;
height: 1px;
background-color: #c8c7cc;
-webkit-transform: scaleY(0.5);
transform: scaleY(0.5);
//房租鼠标点击
pointer-events: none;
}

2:通过stylus预编译器对设备像素比进行检测

首先，新建一个 mixin.styl

border-1px($color)
position: relative
&:after
display: block
position: absolute
left: 0
bottom: 0
width: 100%
border-top:1px solid $color
content: ‘ ‘
//房租鼠标点击
pointer-events: none;

然后再建立一个base.styl,来根据设备dpr进行缩放

@media (-webkit-min-device-pixel-ratio: 1.5),(min-device-pixel-ratio: 1.5)
.border-1px
&::after
-webkit-transform: scaleY(0.7)
transform: scaleY(0.7)

@media (-webkit-min-device-pixel-ratio: 2),(min-device-pixel-ratio: 2)
.border-1px
&::after
-webkit-transform: scaleY(0.5)
transform: scaleY(0.5)

为了方便引用，新建一个index.styl文件统一引入所有styl文件

@import ‘./mixin’
@import ‘./base’

然后以vue项目为例，在main.js中引入

import ‘./common/stylus/index.styl’

随后要在样式的元素中添加 border-1px 类名和在其样式上添加 border-1px(clor颜色值);就可以实现边框在移动端上以1px显示了

3：根据html的font-Size 来使用rem来定义
